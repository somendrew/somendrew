### 🫁 [Pneumonia Detection from X-Rays](https://github.com/somendrew/Chest_XRay-Pneumonia-)

**Deep Learning | CNN | TensorFlow | Streamlit**

CNN-based binary classifier for detecting pneumonia from chest X-rays. Prioritized Recall over accuracy because false negatives in medical diagnosis are dangerous. Deployed as a real-time Streamlit app.

- ✅ Accuracy: **91.3%** | Precision: **92.56%**
- 🔍 Recall: **93.55%** — optimized for medical safety
- 🚀 Live Streamlit deployment
